Aquarius TD

03.07.2014

Each time you build a tower, different moving path!!Action, Strategy Defense!!

Get ready to Defend the Universe!!
Build towers with your limited supply, and Defend the five planets that 26 types of zombies is about to invade.

Each time you build a tower, different moving path.
You have to think fast and have successful strategy and combat them all.

Game Features:
- Include AD.
- There are five planets.(α,β,γ,δ,ε)
- Each planets have story mode and mission mode.
- There are three balance type in the story mode.(normal, hard, master)
- You can buy useful items in the shop.
- You have to defend against monsters during the defined wave.
- You can skip some of the wave in the stage.
- Support auto save(local). Whenever you can continue play.
- If you defend each stage, you can obtain star.(3, 2, 1)
- If you completely defend each stage, you can get a ruby reward.
- Star will be used to open a planet that has been locked.

With Facebook:
- If you login with Facebook, you can play with friends.
- You can request to friends only once a day.
- If you login with Facebook, you can obtain a plus 20% score.
- You can ask your friends for more RUBY or buy a full set of RUBY now.
- If you share on Facebook, will obtain RUBY in game.
